### Prerequisites

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Meti-AVP/scientific-writing-quiz.git
cd scientific-writing-quiz
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
